HOW TO RESUME
-------------
1. Load your API key:
     cd "/Users/aarav/Projects/Home IoT Security"
     set -a && source .env && set +a

2. Run the script (same command every time — it auto-detects the progress file):
     python3 -u "/private/tmp/claude-501/-Users-aarav-Projects-Home-IoT-Security/fee3dd45-ef7c-4364-b735-734f5c24d90d/scratchpad/download_nvd_resumable.py" \
       --api-key "$NVD_API_KEY" \
       --threads 3 \
       --out "/private/tmp/claude-501/-Users-aarav-Projects-Home-IoT-Security/fee3dd45-ef7c-4364-b735-734f5c24d90d/scratchpad/nvd_all.csv"

   On startup it prints:
     - How many CVE IDs are already in nvd_all.csv
     - How many pages are already done (from progress.json)
     - How many pages remain

HOW TO PAUSE
------------
Just kill the process: Ctrl-C in the terminal, or:
     pkill -f download_nvd_resumable.py

Progress is saved to nvd_all.csv.progress.json after EVERY completed page,
so you won't lose more than one page (~2000 CVEs) worth of work.
