# CWE catalog — pinned v4.12

`cwec_v4.12.xml.zip` is the MITRE CWE catalog **v4.12 (June 29, 2023)** — the exact
version of the CWE-888 Software Fault Patterns view used by the transportation IoT
device study (`Onboarding-Docs/transportation_device_study.pdf`, Section III-C),
whose analysis `scripts/cwe888_analysis.py` replicates. Keep it pinned so the
23-primary-cluster mapping stays comparable with the paper.

Source: <https://cwe.mitre.org/data/xml/cwec_v4.12.xml.zip> (downloaded 2026-07-09).
The script reads the XML straight out of the zip — no need to extract it.
